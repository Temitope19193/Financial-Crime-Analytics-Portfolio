# Financial Crime Analytics Portfolio

This portfolio contains three practical projects demonstrating my skills in:

- Fraud detection using machine learning
- Financial crime risk monitoring and dashboarding
- AML network and transaction topology analysis

## Projects

### Project 1: Fraud Detection
Built machine learning models to detect suspicious transactions using Random Forest and XGBoost.

### Project 2: Risk Monitoring Dashboard
Created a Power BI dashboard for monitoring financial crime indicators, alert volumes, and control performance.

### Project 3: AML Network Analysis
Used graph analytics to identify suspicious transaction patterns, mule accounts, and layering behavior. /b
