Train-Test Split¶

The dataset is divided into training and testing sets using stratified sampling to preserve the fraud class ratio.

Model Training¶

An XGBoost classifier is trained to identify fraudulent transactions. Because the fraud class is highly imbalanced, class weighting is incorporated using scale_pos_weight.

In [9]:

    max_depth=6,

    subsample=0.8,
    colsample_bytree=0.8,
    random_state=42,
